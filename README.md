# 简介

本项目将包含常用的 node 模块的使用例子，长期更新。

本人博客：www.zmscode.cn

长期更新 nodejs 和 区块链 相关博文。

## 更新节点

- 初始化项目
- 添加 ESLint 代码规范检测模块，快速上手教程
- 添加 winston 案例
- 添加 pm2 node 案例
- 添加 jsonschema 案例
- 添加 axios Promise 的 HTTP 案例
- 添加 redis 操作
- 添加 console.log 输出文件路径和行号
- 添加 crypto 进行加密和解密
- 添加接口限流设计方案
- 添加 xlsx 文件操作
- 添加 nodemailer 案例
